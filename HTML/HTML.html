Linguagens de programação

(Voltar)

Linguagem de programação é um conjunto de instruções para se gerar um determinado programa

Hoje em dia temos uma variedade de linguagem de programação, e cada uma possui suas proprioas particularidades, mas todas visando o mesmo objetivo que seria o programa/site (software)

As linguagens utilizam o termo "sintaxe" para se referir a escrita de determinada linguagem

Segundo RedMonk em uma pesquisa realizada entre Setembro de 2012 ate Janeiro de 2022, essas são as 5 linguagem de programação mais utilizadas

  1. Javascript
  2. Python
  3. Java
  4. PHP
  5. C#

HTML

(Voltar)

HTML significa HyperText Markup Languague ou Linguagem de Marcação de HiperTexto

O HTML não é considerado uma linguagem de programação, pois ele se trata de um arquivo de marcação

HTML é uma linguagem de marcação utilizada na construção de websites e a inserção de novos conteúdos

Documentos HTML podem ser interpretados por navegadores

Estrutura básica do HTML

(Voltar)

A Estrutura básica do HTML é realizada da seguinte forma

Inicia com o termo html entre os sinais de "maior que" e "menor que"

Logo em suguida temos a tag head que sera o titulo da pagina

Em seguida temos o body que será o corpo da página aonde o usuario final ira visualizar

Tag

(Voltar)

As Tag's são elementos do HTML, utilizadas para informar ao navegador a estrutura do site

As Tag's serão interpretadas pelo navegador produzindo assim o conteúdo da pagina

As Tag's são dividas em dois tipos, as que precisam de fechamento e as qe não precisamde fechamento

Atributos

(Voltar)

Atributos são propriedades que uma Tag pode possuir, possuindo os atributos globais gerais e alguns mais especificos

Com ele podemos identificar melhor um elemento, informar qual arquivo aquela tag deve utilizar, indicar tipo de um campo de texto, etc...
